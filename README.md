# Selenium Login Test

This project automates a login test using Python and Selenium on the [Practice Test Automation](https://practicetestautomation.com/practice-test-login/) website.

## Tools Used
- Python
- Selenium WebDriver
- ChromeDriver

## Steps Automated
- Visit the login page
- Enter credentials
- Click login
- Verify success message
